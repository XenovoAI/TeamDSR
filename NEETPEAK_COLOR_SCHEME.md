# NEETPeak Color Scheme Implementation Plan

## Primary Teal/Turquoise Palette
- **Emerald Dark** (#1B5E5E): Headers, dark backgrounds
- **Atlantis** (#0B9B9B): Primary buttons, links
- **Verdigris** (#0DCDCD): Hover states, active elements
- **Turquoise** (#5DDDDD): Light accents, badges
- **Aero Blue** (#AFFFFF): Very light backgrounds, subtle highlights

## Implementation Strategy

### 1. Global Theme (Tailwind Config)
- Update primary color to teal/turquoise scale
- Replace blue/indigo with emerald-atlantis-verdigris

### 2. Components to Update
- Navbar: Emerald background with turquoise accents
- Buttons: Atlantis primary, Verdigris hover
- Cards: White with turquoise borders
- Badges: Turquoise backgrounds
- Links: Atlantis color

### 3. Page-Specific Updates
- Home: Emerald hero section
- Materials: Turquoise accents
- Mentorship: Atlantis highlights
- Login: Verdigris buttons

### 4. Accent Usage
- Purple: Special features, premium content
- Yellow: Warnings, important notices
- Coral: Errors, delete actions
- Green: Success messages
- Red: Error states
